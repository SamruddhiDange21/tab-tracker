# Tab Tracker

A simple productivity tool that forces you to think before opening a tab.

##  Features
- Track why you open each tab
- Categorize activity (Learning / Entertainment / Other)
- View history in dashboard
- Basic insights on usage patterns
- Clean modern UI

##  Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript



## ▶️ How to Run

```bash
pip install flask
python app.py
